# Trader Behavior Analysis Using Fear & Greed Index

## Overview
This project analyzes how market sentiment (Fear vs Greed) affects trader performance
and behavior using transaction-level trading data and a Fear & Greed Index.

## Data
- historical_data.csv: trader transaction dataset
- fear_greed_index.csv: daily sentiment index

## Methodology
1. Merge trader data with sentiment data by date
2. Classify days into Fear and Greed
3. Compare performance metrics across sentiment regimes
4. Segment traders based on leverage, frequency, and consistency
5. Propose actionable trading strategies
